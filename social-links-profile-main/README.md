### What I learned

- How to structure a list of links using `<ul>`, `<li>` and `<a>` for proper semantic HTML
- How to use `display: block` on an `<a>` element so it fills the full width of its container
- How to remove list bullets with `list-style: none`
- How to use `:hover` and `:focus` pseudo-classes to create interactive button states
- How to open links in a new tab using `target="_blank"`
- How to define and use CSS custom properties (variables) with `:root` and `var()`
- How to convert pixel values to rem units (px ÷ 16 = rem)

### Continued development

- Improve understanding of responsive design and media queries
- Practice more with CSS Flexbox and Grid
- Get more comfortable reading and applying design specifications
